# Mashina.uz — Полный гайд по функционалу

Сайт запущен на **http://localhost:9000**

---

## 1. Главная страница

Открой **http://localhost:9000**

- Автоматически перенаправляет на `/ru`
- Hero-блок с заголовком «Купите или продайте авто в Узбекистане»
- В поисковой форме выбери **марку** → **модель** → **город** → нажми **«Найти»**
- Переходит в каталог с применёнными фильтрами

Прокрути вниз:
- Сетка последних объявлений с карточками
- Блок **«Популярные марки»** — клик на марку открывает каталог с фильтром
- Блок преимуществ и CTA-баннер «Продайте авто быстро»

---

## 2. Каталог объявлений

Открой **http://localhost:9000/ru/listings**

**Карточки объявлений:**
- Фото, цена, пробег, город, год
- Бейдж давности («Сегодня» / «Вчера» / «5 дн. назад»)
- Синий значок щита **«Проверено»** — на машинах, прошедших осмотр
- Кнопка **сердечко** — добавить/убрать из избранного прямо с карточки
- Кнопка **GitCompareArrows** — добавить к сравнению прямо с карточки

**Фильтры (левая панель / мобильный ящик):**

*Основные:*
- **Марка** — выпадающий список
- **Город** — выпадающий список
- **Цена от/до** — числовые поля
- **Год от/до**
- **Тип кузова** — кнопки (Седан, Кроссовер, Хэтчбек…)
- **Топливо** — кнопки (Бензин, Дизель, Газ, Электро, Гибрид)
- **КПП** — кнопки (Автомат, Механика, Вариатор, Робот)
- **Состояние** — Новый / Б/у
- **Сортировка** — новее/дешевле/дороже

*Дополнительные:*
- **Модель** — текстовое поле
- **Пробег от/до** — числовые поля
- **Привод** — кнопки (Передний, Задний, Полный)
- **Объём двигателя от/до** — числовые поля
- **Цвет** — цветные кружки (Белый, Чёрный, Серебро, Серый, Красный, Синий, Зелёный, Другой)
- **Только проверенные** — чекбокс «Верифицировано»

На кнопке «Фильтры» и кнопке «Сбросить» — **бейдж с числом активных фильтров**.

**На мобильном (< lg):**
- Кнопка **«Фильтры»** с бейджем открывает нижний drawer
- Кнопка **«Применить»** закрывает drawer и обновляет список

**Пагинация:** кнопки «Предыдущая» / «Следующая» внизу списка.

**Пустой результат:** показывает CTA-кнопки «Добавить объявление» и «Сбросить фильтры».

**Скелетон-загрузка:** при переходе на страницу пульсирующие плейсхолдеры вместо пустого экрана.

## 3. Страница объявления

Кликни на **любую карточку** в каталоге.

### Галерея
- Стрелки влево/вправо — переключение фото
- **Клавиши ← →** на клавиатуре тоже работают
- Все изображения загружаются лениво (`loading="lazy"`)

### Характеристики
Год, пробег, топливо, коробка, привод, объём двигателя, цвет, кузов, состояние.

### Блок продавца
- Имя продавца — кликабельная ссылка на страницу продавца
- Кнопка **«Показать телефон»**

### Кредитный калькулятор
- Стоимость авто подставлена автоматически
- Измени первоначальный взнос, срок (12–84 месяца), ставку
- Ежемесячный платёж и переплата пересчитываются мгновенно

### Кнопки действий
- **В избранное** (сердечко) — добавить/убрать, иконка меняется
- **К сравнению** — добавить/убрать из сравнения (максимум 3 авто)
- **Поделиться:**
  - Telegram — открывает мессенджер со ссылкой
  - WhatsApp — открывает WhatsApp со ссылкой
  - Скопировать — тост «Скопировано!» на 2 секунды

### Кнопка «Пожаловаться»
- Открывает модальное окно с выбором причины
- Причины: спам, мошенничество, некорректное описание, неверная цена, уже продан, другое
- После отправки — тост-уведомление об успехе

### Отчёт осмотра (если машина прошла проверку)
- Блок **«Отчёт осмотра Mashina.uz»** ниже характеристик
- Общая оценка в %, 7 прогресс-баров по узлам
- Цветные точки по кузовным панелям (зелёный = ок, жёлтый = дефекты)
- Имя инспектора и дата проверки

### Просмотры
- Счётчик просмотров увеличивается **один раз** на устройство за 24 часа (cookie-based)

## 4. Сравнение автомобилей

**Как добавить к сравнению:**
- На карточке в каталоге нажми иконку GitCompareArrows
- На странице объявления нажми «Добавить к сравнению»
- Максимум **3 авто** — при попытке добавить четвёртое появится тост-ошибка

**Плашка сравнения:**
- После добавления первого авто внизу экрана появляется плашка **CompareBar**
- Показывает название авто (марка, модель, год)
- Кнопка «×» убирает авто из сравнения
- Кнопка «Сравнить» открывает страницу сравнения

**Страница сравнения:**
Открой **http://localhost:9000/ru/compare?ids=1,2,3**

- Таблица характеристик: цена, год, пробег, объём, топливо, КПП, привод, кузов, цвет, состояние, город
- Различающиеся значения выделяются визуально
- Кнопка «Очистить сравнение»

## 5. Избранное

Открой **http://localhost:9000/ru/favorites**

**Без авторизации:**
- Добавляй объявления через кнопку сердечка
- Список хранится в localStorage, сохраняется при перезагрузке
- На странице `/ru/favorites` видишь добавленные карточки

**После входа в аккаунт:**
- localStorage-избранное автоматически синхронизируется в базу данных
- Доступно с любого устройства при следующем входе

## 6. Регистрация

Открой **http://localhost:9000/ru/register**

**По телефону:**
- Нажми «По телефону»
- Имя: `Тест Пользователь`
- Телефон: `+998 90 123 45 67`
- Пароль: `pass1234` (минимум 6 символов), подтверждение
- «Создать аккаунт» → редирект на `/ru/profile`

**По email:**
- Нажми «По email»
- Email: `user@example.com` — валидируется формат
- Имя, пароль
- «Создать аккаунт»

> Попробуй зарегистрироваться с тем же телефоном/email — получишь ошибку «уже используется».
>
> Попробуй 6 раз подряд с разными данными — на 6-й попытке ответ 429 «слишком много попыток» (лимит 5/час на IP).

## 7. Вход в аккаунт

Открой **http://localhost:9000/ru/login**

- Логин: телефон или email
- Пароль
- «Войти» → JWT-кука `user_token` на 30 дней

В хедере появился **аватар с буквой имени**. Кликни:
- Профиль
- Мои объявления
- Настройки
- Выйти

> Неверный пароль — ошибка «Неверный логин или пароль».
>
> 11 попыток подряд — 429 «слишком много попыток» (лимит 10/15 мин на IP).

## 8. Профиль пользователя

Открой **http://localhost:9000/ru/profile**

### Вкладка «Мои объявления»
- Список своих объявлений со статусом (На модерации / Активно / Отклонено)
- Кнопка «+ Добавить» → `/ru/sell`
- Миниатюра фото, цена, просмотры

### Вкладка «Избранное»
- Всё, что добавил гостем или авторизованным — синхронизировано
- Клик на карточку → страница объявления

### Вкладка «История просмотров»
- Последние просмотренные объявления (из cookie + localStorage)
- Хранится в localStorage, не требует авторизации

### Вкладка «Настройки»
- Изменить **имя** — сразу отражается в хедере
- Изменить **телефон**, **email**
- Раздел «Сменить пароль»: текущий пароль + новый + подтверждение
- При ошибке загрузки данных — красный баннер с объяснением (не тихий сбой)

## 13. Тост-уведомления

По всему сайту используется единая система уведомлений:
- **Зелёный** — успех (добавлено в избранное, скопировано)
- **Красный** — ошибка (максимум 3 авто для сравнения)
- **Синий** — инфо (убрано из избранного)
- Уведомления появляются снизу справа, исчезают через 3 секунды

## 18. Модерация объявлений

**http://localhost:9000/admin/listings**

**Вкладки:** Все / Активные / На модерации / Отклонённые

1. Переключись на **«На модерации»** — видишь Toyota Camry с шага 9
2. Нажми **«Одобрить»** → статус «Активно», объявление появляется в каталоге
3. Нажми **«Отклонить»** → статус «Отклонено», объявление исчезает из каталога
4. Снова одобри

**Значок «Проверено»:**
- Нажми иконку щита/галочки → переключает `isVerified`
- На карточке в каталоге появляется/исчезает синий значок

**Редактирование объявления:**
- Кликни на строку → страница `/admin/listings/[id]`
- Редактируй поля, меняй статус, удаляй

**Добавить объявление:**
- «+ Добавить авто» → `/admin/listings/new`
- Форма аналогична пользовательской, статус `active` сразу — без модерации

## 20. Полный цикл осмотра

**http://localhost:9000/admin/inspections**

**Вкладки:** Новые / Запланированы / Проверены / Отменены

### Шаг 1 — Найти заявку
Видишь заявку от шага 12 (Тест, Chevrolet Cobalt).

### Шаг 2 — Запланировать
Кликни на заявку → в блоке «Действия»:
- Выбери дату и время → «Запланировать»
- Статус → «Запланировано»

### Шаг 3 — Привязать объявление
- Из выпадающего списка выбери активное объявление (Toyota Camry)
- «Сохранить»

### Шаг 4 — Заполнить отчёт
Нажми «Заполнить отчёт осмотра»:
- **Оценки узлов (1–5):** Двигатель: 5, КПП: 4, Подвеска: 4, Тормоза: 5, Электрика: 3, Салон: 4, Шины: 3
  - Значения вне диапазона 1–5 не принимаются (ошибка валидации)
- **Кузовные панели:** кликни на каждую часть, выбери состояние
- «Участие в ДТП»: нет
- «Пробег подтверждён»: да
- Заметки: `Автомобиль в отличном состоянии`
- Инспектор: `Иван Петров`
- «Сохранить отчёт»
- Объявление автоматически получает статус `isVerified = true`

### Шаг 5 — Проверить на сайте
Перейди на страницу Toyota Camry в каталоге:
- Синий значок «Проверено» на карточке
- Блок **«Отчёт осмотра Mashina.uz»** на странице объявления
- Общая оценка, 7 прогресс-баров, цветные точки кузова, имя инспектора
